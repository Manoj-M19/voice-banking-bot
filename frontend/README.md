## What it does

- 💬 Type a banking question → get a streaming AI response
- 🔊 Arya speaks the response back using browser TTS
- ⚡ Real-time token streaming via WebSocket
- 🏦 Specialized for Indian banking — EMIs, CIBIL, home/personal/car loans
